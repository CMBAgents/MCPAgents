# Getting started

We recommend using Python 3.12. 

```bash
git clone https://github.com/CMBAgents/MCPAgents.git
cd MCPAgents
chmod +x installer.sh
source installer.sh
jupyter-lab
```

You can the run the notebook (make sure you select the mcp_env kernel in jupyterlab).
